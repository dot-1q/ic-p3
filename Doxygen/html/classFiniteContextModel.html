Detailed Description

Finite context Model class used to collect high-order statistical information of an information source, assigning a probability estimate to the symbols of the alphabet, according to a conditioning context computed over a finite and fixed number of past outcomes.

◆ calculateEntropy()

double FiniteContextModel::calculateEntropy ( )

Calculate the entropy of the model.

Returns
double The Entropy value of the model.

Calculates as defined by: Entropy = sum(Hc * Pc). Pc = Probability of the context/sub-model. Hc = Entropy of the context/sub-model. Taking into consideration the alpha and aphabet size.

Returns
double Has to be between 0 and log2(27).
